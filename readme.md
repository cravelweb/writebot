# Ghostwriter

Ghostwriter は、OpenAI API (ChatGPT) を使って WordPressの投稿記事を自動生成する WprdPress プラグインです。

## テーマライセンス

配布物に含まれている PHP、画像、JavaScript、CSS（スキン含む）等のすべてを GPL として公開します（100%GPL）。
