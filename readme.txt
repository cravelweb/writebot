== Description ==

Writebot AI is a WordPress plugin that uses the OpenAI API (ChatGPT) to automatically generate WordPress post content.

This plugin works in Japanese and English WordPress environments.

== Installation ==

1. From the WP admin panel, click "Plugins" -> "Add new".
2. In the browser input box, type "Writebot AI".
3. Select the "Writebot AI" plugin and click "Writebot AI".
4. Activate the plugin.

ORâ€¦

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click "Plugins" -> "Add new".
4. Click "upload".. then browse to the .zip file downloaded from this page.
5. Click "Install".. and then "Activate plugin".



== Usage ==

1. Once installed, 'WriteBot AI settings' will be added to the 'Settings' menu in the WordPress admin pane.
2. In the 'Open AI API KEY' box of the settings screen, please enter the API key 'sk-....' that you have obtained from OpenAI.
3. Select the model you want to use from this list and save the settings again.
4. Content generation is done from the post editing screen. When you open the editing screen from creating a new post, etc., the 'Writebot AI' area will be displayed in the metabox area at the bottom of the screen.

== Frequently Asked Questions ==

= About generated content =

Although there are several setting items, ChatGPT does not strictly adhere to all setting items.
Even if the instructions are correctly communicated to ChatGPT, text may be generated that differs from those instructions.
Think of it as a guideline or loose instruction for content generation.
